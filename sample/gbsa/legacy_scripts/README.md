# `sample/gbsa/legacy_scripts/` — Legacy POC Scripts (pre-1.22.0)

This directory preserves the 4 original Python scripts that drove the
GENESIS MM/GBSA workflow before the `abmptools.genesis.mmgbsa`
sub-package was added in v1.22.0:

| Script | Stage | Replacement |
|---|---|---|
| `1_devidepdb.py` | PDB split (Biopython) | `python -m abmptools.genesis.mmgbsa divide` |
| `2_setupmd.py` | acpype + tleap | `python -m abmptools.genesis.mmgbsa parameterize` |
| `3_rungbsa.py` | atdyn GBSA single-point | `python -m abmptools.genesis.mmgbsa run` |
| `4_analyse.py` | log parser + CSV + plot | `python -m abmptools.genesis.mmgbsa analyze` |

## Why both are kept

- **Reproducibility**: the POC outputs in
  `~/repos/abmptools-sample/sample/gbsa-genesis/` were generated by
  these exact scripts. Keeping them allows bit-exact regeneration.
- **Education**: the scripts show the minimum viable pipeline before
  the abstractions in v1.22.0. Useful when teaching the workflow.

## Differences vs v1.22.0 (mostly ergonomic)

- Hard-coded `atdyn` path (`/home/okuwaki/GENESIS_Tutorials-2022/...`) →
  v1.22.0 uses `atdyn_path` config knob + `mpirun -np N`.
- `os.chdir` side effects → v1.22.0 uses `cwd=` argument to subprocess.
- No JSON config / no validation → v1.22.0 has dataclass + JSON
  round-trip + `validate` sub-command.
- ΔG_bind formula: `4_analyse.py` writes
  `(egas+S)_c - (egas+S)_l - (egas+S)_r` where `egas = ENERGY - SOLVATION`.
  v1.22.0's `compute_dg_bind` writes `E_c - E_l - E_r` directly. Both
  forms are **algebraically identical** (since `egas + S = ENERGY`)
  and produce the same ΔG_bind. Per GENESIS doc `05_Energy.rst:564`,
  the `ENERGY` column is the total potential `U = U_FF + ΔG_solv` —
  SOLVATION is a sub-component, already inside `ENERGY`. v1.22.0 also
  ships `compute_dg_components` for users who want the
  `{dg_mm, dg_solv, dg_bind}` decomposition explicitly.

## Running the legacy scripts

```bash
# Stage 1
python 1_devidepdb.py -i input/3772L-rename.pdb -r 201

# Stage 2 (requires acpype + AmberTools)
python 2_setupmd.py

# Stage 3 (edit hard-coded atdyn path first!)
python 3_rungbsa.py

# Stage 4 (POC ΔG formula -- algebraically identical to v1.22.0)
python 4_analyse.py
```
