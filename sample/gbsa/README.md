# `sample/gbsa/` — GENESIS MM/GBSA Inputs

This directory ships configuration examples for
`abmptools.genesis.mmgbsa` (1.22.0+).

## Files

| File | Purpose |
|---|---|
| `example_config.json` | POC reproduction (4 targets at ligand_resno=201). |
| `smoke_config.json` | Minimal 1-target smoke for CI / sanity checks. |
| `legacy_scripts/` | The 4 original POC Python scripts (`1_devidepdb.py` ... `4_analyse.py`), kept for historical reference. |

## Quick start

```bash
# Validate the config (verifies tools + Python deps).
python -m abmptools.genesis.mmgbsa validate --config example_config.json

# Folder-mode shortcut (POC compatibility):
python -m abmptools.genesis.mmgbsa pipeline -i ./input -r 201 -o ./out

# Config-mode (re-runnable):
python -m abmptools.genesis.mmgbsa pipeline --config example_config.json -o ./out
```

The pipeline produces:

```
out/
├── input/config.json                # frozen config for reproducibility
├── dirnames.in                      # POC compatibility (per-target dirs)
├── <target>/
│   ├── <target>.pdb                 # original copy
│   ├── <target>_receptor_201.pdb    # Stage 1 output
│   ├── <target>_ligand_201.pdb
│   ├── <target>_ligand_201.acpype/  # acpype's GAFF + AM1-BCC
│   ├── leaprc_{complex,ligand,receptor}
│   ├── {complex,ligand,receptor}.{prmtop,inpcrd,pdb}
│   ├── {complex,ligand,receptor}.inp
│   └── {complex,ligand,receptor}.log
└── analysis/
    ├── analysis_results.csv         # ΔG_bind table
    └── dg_bind_plot.png             # bar plot
```

## Input PDBs

The 4 POC PDBs (3772L / 9MM52 / L7759 / M99KZ at ligand_resno=201) are
**not** committed to abmptools (research data). To reproduce, place
the four `*-rename.pdb` files into `sample/gbsa/input/` (or any
directory referenced by `input_dir`) before running.

The smoke config expects a single `smoke.pdb` for connectivity testing;
provide any small protein-ligand complex (or use the synthetic input
generated by `tests/test_mmgbsa_integration.py`).
